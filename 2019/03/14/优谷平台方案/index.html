优谷线上服务平台方案

线上服务平台目前主要涵盖四个方面：

  1. 网站平台
  2. 优谷在线商城
  3. App
  4. 微信公众号

网站平台

网站平台作为优谷农业的脸面，需要做到恰到好处的形象展示和产品+平台+内容的推荐。

App平台

目前只考虑两个平台：iOS 与 Android，Windows Phone 及 Windows 平台暂不考虑。

App功能

先说明，建议App不要单独做成商城！不要做成淘宝网那样。建议整合线上线下功能。

基本功能

如图所示就行：

一、线上商城功能：
  1. 账号：建议直接使用手机号+验证短信模式登录，跟ofo登录方式一样。一来，便于满足国家对于网络购物的实名制要求，直接通过手机号确认身份。而来，方便用户找回密码，减少死账号的数量，降低数据库压力。最后，有利于增强安全性，减少撞库风险！！
  2. Cookies：记住账号的相关信息即可，比如购买了但未保存送货地址，通过记忆可以免于用户重新输入，增加用户体验。
  3. 商城页面：商城页面仅需单独拿一个页面即可（如下图所示）。
  4. 搜索功能和条件筛选功能，没有这个功能就跳楼啊！！

当然这也是示意，下面说说商城具有的特色功能：

商城特色功能

  1. 以菜名方式选择，用户可以通过选择自己要做的菜名，自动进行选择食材。我想了想，荤菜不太方便，比如我要做宫保鸡丁，鸡丁还得顾客自己买？？能不能整合一下，这个问题考虑一下。

  2. 深度学习

    深度学习推荐菜品：根据用户的口味，买的果蔬，推测顾客爱吃的蔬菜和口味，进行智能推荐；还可根据用餐人数，自动推荐购买数量；

    深度学习推荐周菜单：对于忙的上班族，可直接生成一周菜单，生成菜单后，顾客可以进行调整，确认菜单，直接与顾客进行一周的送菜服务，还可以外送打印做菜方法；

    深度学习送菜方式：根据用户的下班时间，推测最佳送菜时间；

    深度学习购买量：根据用户一段时间的购买情况，推测人数，并建议购买量。

  3. 礼品卡功能

好友、亲人之间可以赠送一定面值的礼品卡，直接用于消费。不解释，星爸爸这点做的非常好：

  1. 线下自提

各类便利店这些年也很普及了，与便利店合作，为附近小区的顾客提供自提服务，再顾客下班前的约30～60分钟，将新鲜果蔬送至便利店，顾客回家直接自提即可。很重要的一点，客户自提是降低成本的一个很好的方法，可以一次性解决一片区域的送菜问题。为了鼓励顾客自提，可给自提顾客随产品赠送优谷纪念品，例如优谷环保手提袋：

一来，减轻配送压力；二来，降低配送成本，避免交通拥堵造成送货不及时；第三，好看的、高质量的纪念品，有利于顾客携带使用，帮助推广产品。很丑的纪念品还不如不送，浪费了钱💰，还会被顾客扔掉。

  1. 线下自提功能，必然就要有提货门店地图功能，必须的！！
  2. 等级功能，这个得用好了，用不好流失用户；用得好才能保住用户，这个得想想。

二、整合线下功能：

  1. 在线下自提地图中，也要整合优谷农场的位置，便于用户驾车前往优谷农场，参与活动。
  2. 在线预约功能，免费农场毕竟是有限的，为了提高用户体验，建议增加预约功能，比如我预约再本周六上午来农场。

总之体验很不好就行了！小程序的体验就如行云流水一般☁️，丝滑般极致体验。

什么是小程序？

​ 小程序与传统App不同，传统App是基于操作系统的、已离线的可执行代码程序；而小程序是基于服务器的、可在线使用的、可部分离线的可执行代码程序。

微信公众号
